# AI Q&A Chatbot with LangChain and Groq API

## Overview

This project is an AI-powered chatbot built using LangChain, Groq API, and Streamlit.

The chatbot accepts user questions and generates intelligent responses using Groq's Llama 3.1 model.

## Technologies Used

- Python
- Streamlit
- LangChain
- LangChain Core
- LangChain Groq
- Python Dotenv

## Files Included

- langchain60_app.py
- requirements_1.txt
- Langchain_groqAPI_chatbot_synopsis.docx
- LangChain_chatbot_streamlit_output.pdf

## Installation

pip install -r requirements_1.txt

## Run

streamlit run langchain60_app.py

## Features

- Real-time Question Answering
- Chat History
- LangChain Integration
- Groq LLM Integration
- Streamlit UI
- Secure API Key Handling
